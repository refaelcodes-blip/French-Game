Corner the Blue

A small French abstract strategy game implemented in C# and Windows Forms.

Corner the Blue is a minimalist implementation of a traditional French hunt game: three slower pieces advance together to block a single, more agile runner.

The red side controls three pieces that can move one step forward or sideways, but never backward. The blue side controls a single piece that can move one step in any direction. Red wins by completely blocking the blue piece so it has no legal moves. Blue wins the moment it passes below all red pieces and breaks through the line.

Desktop implementation in C# / .NET Windows Forms · three red tokens advancing against one blue runner.

Rules at a glance

  • Two players: Red (3 pieces) vs Blue (1 piece).
  • Red moves first; players alternate turns.
  • Red pieces move one step forward or sideways, never backwards.
  • The blue piece moves one step in any direction.

How to win

  • Red wins by trapping the blue piece so it has no legal moves.
  • Blue wins by moving to a position below all red pieces (breaking through the line).
  • No captures, no luck – just pure positional play.

Implementation

  • Implemented in C# on .NET Windows Forms.
  • Board drawn with standard WinForms painting (GDI+).
  • Mouse input for selecting and moving pieces.
  • Compact game logic: great as a learning project or AI testbed.
